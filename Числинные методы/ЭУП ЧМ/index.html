ЭЛЕКТРОННО-МЕТОДИЧЕСКОЕ ПОСОБИЕ
ПО ЧИСЛЕННЫМ МЕТОДАМ
ДЛЯ СТУДЕНТОВ НАПРАВЛЕНИЙ МАТЕМАТИКА, ПРИКЛАДНАЯ МАТЕМАТИКА И ИНФОРМАТИКА

Студент изучающий численные методы должен знать и уметь:

   • основные методы решения систем линейных алгебраических уравнений и нелинейных уравнений;
   • методы приближения функций одной переменной, включая интерполяцию и среднеквадратичное приближения;
   • основные квадратурные формулы и формулы численного дифференцирования;
   • численные и численно-аналитические методы решения задачи Коши для обыкновенных дифференциальных уравнений;
   • применять методы для решения систем линейных алгебраических уравнений и нелинейных уравнений;
   • строить интерполяционные многочлены, элементы наилучшего среднеквадратического и равномерного приближений, интерполяционные сплайны;
   • применять квадратурные формулы для вычисления интегралов, включая неопределенные;
   • решать с помощью одношаговых и многошаговых методов задачу Коши для обыкновенных дифференциальных уравнений;
   • строить и применять методы решения краевых задач для обыкновенных дифференциальных уравнений;
   • составлять алгоритмы и программы для решения основных задач вычислительной математики на ПЭВМ;
   • проводить вычислительный эксперимент с применением стандартных методов для решения более сложных задач математического моделирования.
