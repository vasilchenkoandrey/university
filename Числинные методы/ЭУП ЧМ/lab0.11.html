ЛАБОРАТОРНАЯ РАБОТА 11
МЕТОД РУНГЕ – КУТТА РЕШЕНИЯ ЗАДАЧИ КОШИ ДЛЯ ОБЫКНОВЕННОГО ДИФФЕРЕНЦИАЛЬНОГО УРАВНЕНИЯ

Требуется найти приближенное решение уравнения

y´ = f ( x , y ),   x ∈ [ x0, X ],     (11.1)

при заданном начальном условии

y ( x0 ) = y0.     (11.2)

Метод Рунге – Кутта решения задачи ( 1 ), ( 2 ) относится к одношаговым методам и дает возможность вычислить приближенное решение поставленной задачи {yi } на сетке {xi } ( i = 1,2,..,n ) по формуле :

yi+1 = yi + Δyi,   i = 0,1,..,n-1,     (11.3)

здесь Δyi - приращение, вид которого для явных методов Рунге -Кутта различного порядка точности вычисляется следующим образом:

1.Метод первого порядка точности:

Δyi = h * f ( xi ,yi );   h = xi+1 - xi.

2. Метод второго порядка точности:

Δyi = h * f ( xi + h/2, yi + k1/2 );

k1 = h * f ( xi , yi );

3. Метод второго порядка точности:

Δyi = (k1 + k2 ) / 2;

k1 = h * f ( xi , yi );   k2 = h * f ( xi + h, yi + k1 );

4.Метод третьего порядка точности:

Δyi = ( k1 + 3k3 ) / 4;

k1 = h * f (xi ,yi);   k2 = h * f ( xi + h/3, yi + k1/3 );

k3 = h * f ( xi + 2h/3, yi + 2k2/3 );

5.Метод третьего порядка точности:

Δyi = ( k1 + 4k2 + k3) / 6;

k1 = h * f ( xi , yi );   k2 = h * f ( xi + h/2, yi + k1/2 );

k3 = h * f ( xi + h, нi+ k1 + 2k2);

6.Метод четвертого порядка точности:

Δyi = ( k1 + 4k3 +k4 ) / 6;

k1 = h * f ( xi , yi );   k2 = h * f ( xi + h/4,yi + k1/4 );

k3 = h * f ( xi + h/2, yi + k2/2);   k4 = h * f ( xi + h, yi + k1 - 2k2 + 2k3);

7. Метод четвертого порядка точности:

Δyi = ( k1 + 2k2 + 2k3 + k4) / 6;

k1 = h * f ( xi , yi );  k2 = h * f ( xi + h/2, yi + k1/2 );

k3 = h * f (xi + h/2, yi + k2/2);  k4 = h * f ( xi + h, yi + k3);

8. Метод четвертого порядка точности:

Δyi = ( k1 + 3k2 + 3k3 + k4) / 8;

k1 = h * f ( xi , yi);  k2 = h * f (xi + h/3, yi + k1/3 );

k3 = h * f ( xi + 2h/3, yi + k1/3 + k2);

k4 = h * f ( xi + h, yi + k1 - k2 + k3);

 

Замечание 1.Отладку программы, реализующей метод Рунге – Кутта, рекомендуется проводить, взяв в качестве теста задачу, решение которой можно получить в аналитическом виде, например:

1. y´ = - x · y ,   x ∈ [ 0, 2 ],

y (0 ) = 1.

2. y´ = x - y,   x ∈ [ 0, 2 ],

y (0 ) = 1.

Вычисления можно проводить на равномерной сетке

xi = i h,  i = 0,1,..,n,   h = ( X - x0 ) / n.

Замечание 2. При реализации метода Рунге – Кутта с заданной точностью ε используется формула Рунге приближенной оценки погрешности

r = (y1i+1 - y2i+1 ) / ( 2p - 1 ),     (11.4)

здесь y1i+1 - решение, вычисленное в точке xi+1 при движении из точки xi с шагом h, y2i+1 - решение, вычисленное в точке xi+1 при движении из точки xi с шагом h/2, p - порядок точности используемой формулы.

Если выполняется условие

| r | < ε ,     (11.5)

то решение в точке xi+1 можно считать вычисленным с заданной точностью и, полагая yi+1 = y2i+1 , переходить к вычислению решения в следующей точке с шагом h ; если , более того, | r | < ε /2 , то шаг можно удвоить h = 2h . При невыполнении условия ( 5 ) полагается h = h/2, и повторяются предыдущие вычисления ( 4 ) .
