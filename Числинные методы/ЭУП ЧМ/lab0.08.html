Аналогичные расчётные формулы можно получить, используя иной подход.

 

Распространенным методом минимизации функции большого числа переменных является метод градиентного спуска. Последующее приближение в этом методе получается из предыдущего смещением в направлении градиента функции F(x), т.е. каждое последующее приближение ищется в виде:

 

xk+1 = xk - bkgradF(xk) (3.1)

 

Параметр bk можно определить из условия минимума величины

 

F(bk) = F(xk - bkgrad F (xk)),

 

рассматриваемой как функции от bk. При таком выборе параметра bk метод называется методом наискорейшего спуска.

 

Пусть дана система уравнений

Ax = b (3.2)

 

где A = AT ( A - симметричная, положительно определенная матрица ), тогда задача решения системы (1.1) эквивалентна отыскания минимума функции

 

F(x) = (Ax, x) - 2(b,x).

 

Покажем это: действительно,

grad F(x) = 2(Ax - b); (3.3)

 

необходимое условие экстремума функции F(x) (в данном случае оно одновременно является и достаточным) имеет вид:

 

grad F(x) = 0

или, согласно (3.3):

Ax - b = 0

 

 

Реализация решения СЛАУ в этом случае состоит в следующем:

1. Задается начальное приближение - вектор x0.

2. Вычисляется следующее приближение xk+1 по формуле

 

xk+1= xk - bkrk ,

 

где вектор rk - невязка, rk = Axk - b,

 

bk = (rk ,rk) ∕ (Ark,rk ).

 

3. Для контроля точности полученного приближения вычисляется величина

 

E =║ xk+1 - xk ║ =│bk │║ rk ║.

 

Если E < ε , где ε - заданная точность, то итерационный процесс прекращается.
